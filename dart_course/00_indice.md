Questo corso ti insegna Dart partendo dalle basi, usando come esempi concreti il codice reale
del progetto **Apiary App** (un'app Flutter per la gestione di apiari).

## Come Usare Questo Corso

1. Segui i moduli in ordine — ognuno si basa sui precedenti.
2. Ogni lezione mostra prima il **concetto puro** in Dart, poi **dove trovarlo** nel progetto reale.
3. I percorsi dei file sono relativi a `lib/`.
4. Dopo ogni modulo trovi **esercizi pratici** da svolgere esplorando i file del progetto.

---

## Prerequisiti

- Nessuna esperienza di Dart richiesta.
- Conoscenza base di un qualsiasi linguaggio di programmazione (anche Python o JavaScript).
- Avere il progetto aperto in un IDE (VS Code o Android Studio).
